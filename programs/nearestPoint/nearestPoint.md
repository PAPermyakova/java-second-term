# Описание

Найти такую точку заданного на плоскости множества точек, сумма расстояний от которой до остальных минимальна.
***
## Входные данные:
Первое число - количество различных точек
Двойка чисел (x и y) координаты точки

## Выходные:
Пара чисел
***
Пример:
Входные
5
0 2
2 0
0 1
1 0
0 0

Выходные
0 0
